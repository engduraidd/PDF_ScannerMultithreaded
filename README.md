Application Name: PDF Scanner Multithreaded
Group name: ThreadX2
Company: American University of Iraq (AUIS)

Overview:
This application was developed by a group of two students as an assignment for the Parallel Programming course (SE422) at the American University in Iraq.


Program concept:

The user is asked to enter a file address on the computer, and the number of PDF files in the file is calculated using three methods
1- Single Thread
2- 4 Threads (MultiThreads)
3- Thread Pool

With make sure for all threads to be thread-safe by avoiding problems like "Race conditions" using mutex (locks)
